## Featured Projects

### 🤖 Will-U-Code — AI Coding Agent System
> *Private · TypeScript · Go · React · Docker · MCP*

A full-stack autonomous coding platform built from scratch. The system is composed of four main layers:

- **UI** — A React-based interface for submitting development tasks and inspecting agent output in real time
- **API** — A backend service that receives requests, manages state, and coordinates communication between components
- **Orchestrator** — The brain of the system; routes tasks between agents, manages context windows, handles bidirectional communication between the API layer and agents, and enforces execution order
- **Agents** (Go) — Autonomous workers capable of planning tasks via RAG, generating and linting code, and reviewing output against quality checklists. Each agent supports **Skills** (reusable capability modules) and **MCP** (Model Context Protocol) for tool use and external integrations, as well as **sub-agents** for parallelized subtask execution

The system supports three core actions: `plan` (decompose a user request into tasks), `code` (generate and lint code for a given task), and `review` (assess code quality). All output is structured for downstream parsing, making it suitable for integration into GitHub Actions workflows.

**Architecture highlights:** Docker-based deployment · Bidirectional API ↔ Agent communication · 50-line file modularity constraint · Strict TypeScript + Go typing · >80% test coverage target
